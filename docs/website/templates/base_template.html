The easy to use creative coding library for Python

LineDream makes creating generative art simple and intuitive.
Output SVGs for the web or for pen plotters.

View on GitHub Tutorials API Documentation 
from LineDream import Canvas, Rectangle

Canvas.width = 300
Canvas.height = 300
Canvas.background_color='white'

for x in range(18):
	Rectangle(150,150, 200, 200).rotate(10*x)

Canvas.save("rectangles.svg")
